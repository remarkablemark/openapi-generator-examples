# Fern

[Documentation](https://github.com/fern-api/fern)

## Prerequisites

[Node 18+](https://nodejs.org/):

```sh
brew install node
```

[Docker](https://www.docker.com/):

```sh
brew install docker
```

## Install

Install the CLI:

```sh
npm i
```

## Init

Initialize Fern with your OpenAPI spec:

```sh
npm run init
```

## Generate

Generate the TypeScript SDK with Docker:

```sh
npm run generate
```
